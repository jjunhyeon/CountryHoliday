
## 1. 프로젝트 개요 및 환경
- 언어 Java 21
- 프레임워크 Spring Boot 3.4
- 빌드 도구 Gradle
- DB H2 Database

<br>
<br>

## 2. 빌드 및 실행 방법

1. IDE에서 프로젝트 클론 또는 BASH 환경에서 git clone [원격 저장소 URL] 
2. cd [프로젝트 폴더명]
3. 빌드 : IDE 직접 GUIT GRADLE BUILD 수행 / BASH - [./gradlew build]
4. 실행
3-1. IDE 에서 BOOT DASHBOARD에서 선택 실행
3-2  BASH 환경에서 ./gradlew bootRun
